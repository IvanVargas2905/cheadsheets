## Estructura de un documento HTML
Un documento HTML básico consta de las siguientes partes:
```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Nombre de tu página</title>
</head>
<body>
    <!-- Contenido de tu página va aquí -->
</body>
</html>

<head>
    <meta charset="UTF-8">

    <script src="script.js"></script>
</head>
